# 🐹 Hamster Wheel Loader

A beautiful and modern **Hamster Wheel Loader** built using **HTML & CSS**. This project features a realistic hamster running inside a spinning wheel, accompanied by animated loading text, a glowing progress bar, and a vibrant gradient background.

Perfect for websites, dashboards, portfolios, and web applications that need an engaging loading screen.
